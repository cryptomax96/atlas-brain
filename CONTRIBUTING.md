## How to Contribute

### 1. Fork the Repository

```bash
gh repo fork cryptomax96/atlas-brain
git clone https://github.com/YOUR_USERNAME/atlas-brain.git
```

### 2. Create a Branch

```bash
git checkout -b docs/your-contribution
```

### 3. Make Your Changes

Follow the templates in [TEMPLATES.md](./TEMPLATES.md)

### 4. Submit a Pull Request

```bash
git add .
git commit -m "Add: [Domain] Your contribution"
git push origin docs/your-contribution
gh pr create
```
